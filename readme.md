# セットアップのスクリプト

## setup.sh
+ aptのインストース
+ emacsの設定ファイル
+ Latexのインストール
+ Dropboxのインストール

## 実行
```
$ wget https://github.com/sheltie03/SetUp/archive/master.zip
$ unzip master.zip
$ cd ~/SetUp-master/
$ ls
emacs.txt	moji-bake.txt	setup.sh
mac-emacs.txt	pdfplatex.sh readme.md
$ chmod 755 setup.sh
$ sudo ./setup.sh
```
